# Automate Celo Blockchain Operations with Our New n8n Community Node

We're excited to announce the release of n8n-nodes-celo, a powerful new community node that brings Celo blockchain functionality directly into your n8n workflows. Developed by Velocity BPA, this node opens up endless possibilities for automating blockchain operations without writing a single line of code.

## The Challenge of Blockchain Integration

Integrating blockchain functionality into business processes has traditionally required deep technical expertise and custom development. Whether you're managing stablecoin transactions, handling identity attestations, or participating in governance, the complexity of blockchain APIs creates barriers for teams wanting to leverage Web3 technology.

## Introducing n8n-nodes-celo

Our new Celo node bridges this gap by providing a no-code interface to interact with the Celo blockchain ecosystem. Celo is a mobile-first blockchain platform focused on making financial tools accessible to anyone with a smartphone, and now you can integrate these capabilities seamlessly into your automation workflows.

## Key Features

The n8n-nodes-celo brings comprehensive Celo functionality to your fingertips:

- **Stablecoin Operations**: Automate transactions with cUSD, cEUR, and other Celo stablecoins
- **Identity Attestations**: Manage phone number verification and identity services
- **Staking Management**: Monitor and manage your CELO staking positions
- **Governance Participation**: Automate voting and proposal tracking
- **Mento Integration**: Interact with Celo's stability mechanism

Whether you're building financial applications, managing DAO operations, or creating automated compliance workflows, this node provides the building blocks you need.

## Getting Started

Installation is straightforward. Simply run:


npm install n8n-nodes-celo


Once installed, the Celo node will appear in your n8n instance, ready to be dragged into your workflows. Connect it with other nodes to create powerful automation sequences that span Web2 and Web3.
